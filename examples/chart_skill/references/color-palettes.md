# 配色方案

## 推薦配色

### 方案一：專業藍色系

```
#2c3e50 - 深藍灰 (主色)
#3498db - 亮藍 (輔助色)
#95a5a6 - 灰色 (中性色)
#ecf0f1 - 淡灰 (背景色)
```

### 方案二：自然綠色系

```
#27ae60 - 綠色 (主色)
#2ecc71 - 亮綠 (輔助色)
#f39c12 - 橙色 (強調色)
#e74c3c - 紅色 (警告色)
```

### 方案三：暖色系

```
#e74c3c - 紅色 (主色)
#f39c12 - 橙色 (輔助色)
#d35400 - 深橙 (加強)
#c0392b - 深紅 (重點)
```

## 使用建議

- 選擇對比度足夠的顔色組合
- 考慮色盲使用者，避免僅靠顏色傳遞資訊
- 提供文字標註作為備援說明
