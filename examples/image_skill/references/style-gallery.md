# 風格參考手冊

## 寫實攝影 (Photorealistic)

**描述:** 真實世界的照片級畫質

**範例 Prompt:**
- "professional product photography, soft lighting, clean background"
- "portrait photography, natural lighting, 85mm lens, bokeh"
- "architectural photography, wide angle, golden hour"

**適用:** 產品照、人物寫真、建築攝影

## 數位插畫 (Illustration)

**描述:** 平面設計感的插畫風格

**範例 Prompt:**
- "flat illustration, vector art, vibrant colors, clean lines"
- "editorial illustration, conceptual art, modern style"
- "children book illustration, whimsical, colorful"

**適用:** 文章插圖、社交媒體、教育內容

## 手繪 (Sketch)

**描述:** 手繪風格，花筋線條

**範例 Prompt:**
- "pencil sketch, monochrome, artistic drawing"
- "ink drawing, crosshatch, detailed sketch"
- "watercolor sketch, loose style, artistic"

**適用:** 草圖、概念圖、藝術作品

## 3D 渲染 (3D Render)

**描述:** 電腦生成的 3D 畫面

**範例 Prompt:**
- "3d render, blender, octane render, professional"
- "isometric 3d, clean design, soft shadows"
- "3d character, stylized, game art"

**適用:** 產品展示、遊戲資產、建範渲染

## 動漫 (Anime)

**描述:** 日本動畫風格

**範例 Prompt:**
- "anime style, studio ghibli, cel shading"
- "manga style, black and white, detailed line art"
- "chibi style, cute, colorful"

**適用:** 卡通、角色設計、網路文化

## 極簡 (Minimalist)

**描述:** 簡潔的設計風格

**範例 Prompt:**
- "minimalist design, flat colors, geometric shapes"
- "scandinavian design, clean, functional"
- "bauhaus style, simple, bold"

**適用:** 簡報、網頁設計、品牌識別

## 風格組合建議

### 商業場景
- 企業簡報 → illustration + minimalist
- 產品展示 → photorealistic 或 3d-render
- 品牌形象 → illustration + anime

### 個人場景
- 社交媒體 → illustration + anime
- 藝術創作 → sketch + photorealistic
- 教育內容 → illustration + minimalist
