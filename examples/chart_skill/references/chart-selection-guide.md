# 圖表選擇指南

## 決策樹

```
您想表達什麼？
    │
    ├───→ 比較數值 → 柱狀圖 (bar)
    │
    ├───→ 顯示趨勢 → 折線圖 (line)
    │
    ├───→ 顯示分佈 → 直方圖 (histogram) / 散佈圖 (scatter)
    │
    └───→ 顯示關係 → 散佈圖 (scatter)
```

## 各類型詳解

### 柱狀圖 (Bar Chart)

**最佳用途:** 比較不同類別的數值

**優點:**
- 簡單直觀
- 易於比較
- 支援水平/垂直方向

**缺點:**
- 類別過多時會擁擠
- 不適合顯示時間趨勢

### 折線圖 (Line Chart)

**最佳用途:** 顯示時間趨勢

**優點:**
- 清晰顯示變化趨勢
- 支援多條線比較

**缺點:**
- 數據點過少時效果不佳
- 線條可能造成誤導

### 散佈圖 (Scatter Plot)

**最佳用途:** 探索變數關係

**優點:**
- 顯示變數間的關係
- 可發現離群值

**缺點:**
- 需要一定數據量才有意義
- 讀取較難

## Tufte 原則提醒

1. **最大化 Data-ink Ratio**
   - 移除不必要的裝飾
   - 避免 3D 效果

2. **使用直接標籤**
   - 在數據點上直接標註數值
   - 減少讀者的轉換成本

3. **選擇合適的精度**
   - 不要為了美觀而增加不必要的元素
